Este plugin ofrece la funcionalidad para servir campañas de publicidad nativa en tu blog de WordPress.

== Description ==

Este plugin permite a los publishers de <a href="http://native.ad" title="integración publicidad nativa" target="_blank">nativead</a> publicar campañas de publicidad native en sus  blogs de wordpress sin necesitar de programar ni una sola línea de código.

Instalando el plugin de native<b>ad</b> en tu site puedes empezar a publicar campañas de publicidad nativa en muy pocos minutos. Puedes integrarte automáticamente con el ad server de native<b>ad</b>  con un único click en el menú del plugin. El plugin incluye el tag javascript de native<b>ad</b> en el header de tu site.

== Screenshots ==

1. Busca el plugin en el directorio de WordPress.
2. Fácil de instalar.
3. Incluye el widget en area sidebar de tu template.


== Installation ==

- Accede al panel de administración de tu site.
- Accede al a sección de plugings Plugins> Nuevo 
- Haz click "buscar" y busca “nativead”
- Haz click en el botón "Instalar" 
- Haz click en "Activar Plugin" para activar el plugin de nativead.
